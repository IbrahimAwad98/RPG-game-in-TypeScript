# RPG-spel i TypeScript

Detta är ett enkelt RPG-spel byggt med TypeScript. Utforska världen, slåss mot fiender och upplev ett klassiskt speläventyr direkt i webbläsaren.

## Funktioner
- Skrivet helt i TypeScript
- Slumpmässiga strider med olika fiender
- Samla föremål och förbättra din karaktär
- Enkla kontroller och grafik

## Kom igång

1. Klona repot:
   ```bash
   git clone https://github.com/IbrahimAwad98/RPG-game-in-TypeScript.git
   ```

2. Installera beroenden:
   ```bash
   npm install
   ```

3. Bygg och starta spelet:
   ```bash
   npm run build
   npm start
   ```

4. Öppna din webbläsare och gå till `http://localhost:3000` för att börja spela.
